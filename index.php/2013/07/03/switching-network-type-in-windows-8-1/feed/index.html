Hi Camira,

If you are in Windows 8.1 final builds, when you navigate to PC Settings > Network, by default you should find yourself on the “Connections” tab. If not, click/tap on “Connections” from the left panel, and all applicable connections would be displayed on the right portion of the screen. Click/Tap on the network name (for LAN connections, this may be named as “Network”, “Network 2”, and so on.)

This should bring up the “Find devices and content”. Hope it helps!
